## 📋 残作業一覧

### 🎯 Phase 3.7: 集中度分析システム
**優先度**: High  
**推定工数**: 8-10時間  
**ファイル**: `/nix/common/context/detection/focus.nix`

#### 実装予定機能
- **作業パターン分析**
  - キーボード・マウス活動測定
  - アプリケーション使用時間追跡
  - タスク切り替え頻度分析
  - 深い集中時間の検出

- **中断測定システム**
  - 通知・アラート頻度測定
  - 外部中断要因の特定
  - 中断からの復帰時間計測
  - 中断パターンの学習

- **Flow状態検出**
  - 長時間の連続作業検出
  - エラー率低下の観測
  - キーストローク一定性分析
  - 最適パフォーマンス状態判定

- **疲労度推定**
  - 作業時間累積による疲労計算
  - エラー率増加による疲労検出
  - 時間帯別疲労パターン学習
  - 休憩提案タイミング計算

#### 予定コマンド
```bash
context-analyze-focus      # 集中度分析・測定
context-detect-flow        # Flow状態検出
context-measure-fatigue    # 疲労度測定
context-track-interrupts   # 中断要因追跡
```

#### 技術仕様
- **データ収集**: 1分間隔でのアクティビティサンプリング
- **プライバシー**: 集約データのみ保存、個別アクションは記録しない
- **機械学習**: 個人パターン学習アルゴリズム
- **アラート**: 疲労検出時の自動休憩提案
