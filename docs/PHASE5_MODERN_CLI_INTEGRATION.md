# Phase 5: モダンCLI/TUIツール統合計画

## 概要
ターミナル環境構築ガイド2025年版に基づく、モダンツール群の統合計画

## 対象ツール群

### 高優先度（即効性が高い）
1. **eza** (ls の置き換え) - Git統合、アイコン表示
2. **bat** (cat の置き換え) - シンタックスハイライト
3. **ripgrep** (grep の置き換え) - 超高速検索
4. **fd** (find の置き換え) - 直感的な検索
5. **zoxide** (cd の置き換え) - AI的なディレクトリ移動

### 中優先度（ワークフロー強化）
6. **lazygit** - Git TUIクライアント
7. **yazi** - モダンファイルマネージャー
8. **bottom** - システムモニター
9. **atuin** - 高機能コマンド履歴

### 低優先度（特定用途）
10. **duf** (df の置き換え) - ディスク容量表示
11. **gdu** (du の置き換え) - ディスク使用量分析
12. **fastfetch** (neofetch の置き換え) - システム情報表示

## アーキテクチャ方針

### 既存環境との統合
- **Nix packages**: メインパッケージ管理
- **Home Manager**: ユーザー設定管理
- **Shell aliases**: 従来コマンドの置き換え
- **Neovim integration**: エディタ連携

### プラットフォーム対応
- macOS (主要開発環境)
- Linux (NixOS)
- WSL (Windows開発)
- Android (Termux) - 軽量版のみ

## 実装戦略

### Step 1: パッケージ定義
新しいモジュール `nix/common/tools/modern-cli.nix` を作成

### Step 2: 設定ファイル統合
各ツールの設定を `configs/` に追加

### Step 3: シェル統合
`configs/shell/zsh/` にエイリアスとプラグイン設定

### Step 4: Neovim統合
`configs/editors/nvim/lua/` にTUIツール連携設定
